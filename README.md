# Flight Simulator OFC V2

Versão melhorada em JavaScript + Three.js, pronta para GitHub Pages.

## Arquivos

- index.html
- style.css
- main.js

## Novidades

- avião visualmente melhorado
- prédios com janelas
- aeroportos mais detalhados
- pistas com luzes
- terminal e torre de controle
- estradas na cidade
- mapa 6 km x 6 km
- árvores, montanhas, nuvens e oceano
- física com sustentação, arrasto, gravidade, stall e pouso
- dano estrutural
- explosão ao bater forte no chão
- missões de aeroporto para aeroporto
- HUD completo
- FPS/qualidade ajustável

## Como subir

Envie estes arquivos para a raiz do GitHub:

- index.html
- style.css
- main.js

Depois ative:

Settings → Pages → Deploy from a branch → main → /(root)

## Controles

- W/S: potência
- Setas: pitch e roll
- A/D: yaw
- C: câmera
- Q: qualidade
- R: reset
- N: missão aleatória
- M: próxima missão
