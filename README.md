# Flight Simulator OFC Ultra Stable

Versão estável do jogo em um único arquivo `index.html`.

## Como usar

Envie apenas `index.html` para o GitHub Pages.

## Controles

- W/S: potência
- Setas: pitch/roll
- A/D: yaw
- C: câmera
- Q: qualidade
- R: reset
- N: missão aleatória
- M: próxima missão

## Recursos

- Mapa 5 km x 5 km
- 4 aeroportos
- Missões de aeroporto para aeroporto
- Física com sustentação, arrasto, gravidade e stall
- FPS melhorado
- Gráficos leves
- Cidade, árvores, nuvens, montanhas e oceano
