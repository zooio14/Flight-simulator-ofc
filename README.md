# Flight Simulator OFC — JavaScript Edition

Jogo feito com HTML, CSS, JavaScript e Three.js.

## Arquivos

- index.html
- style.css
- main.js

## Como rodar no GitHub Pages

Envie os três arquivos para a raiz do repositório e ative:

Settings → Pages → Deploy from a branch → main → /(root)

## Controles

- W/S: potência
- Setas: pitch/roll
- A/D: yaw
- C: trocar câmera
- Q: trocar qualidade
- R: resetar
- N: missão aleatória
- M: próxima missão

## Recursos

- mapa 5 km x 5 km
- 4 aeroportos
- missões de aeroporto para aeroporto
- física de sustentação, arrasto, gravidade e stall
- FPS otimizado com instancing
- cidade, árvores, nuvens, montanhas e oceano
