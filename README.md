# Flight Simulator 1 v0.5

Versão com aviões e prédios modelados no código.

## Novidades

- Cessna 172 modelado
- Boeing 737 modelado
- Caça estilo F-16 modelado
- Menu de seleção de avião
- Troca de avião no jogo
- Prédios com janelas, vidro, telhado e antenas
- Cidade mais realista
- Texturas procedurais
- Mundo aberto sem paredes invisíveis

## Controles

- W/S: potência
- Setas: voar
- A/D: leme
- C: câmera
- R: reset
- 1: Cessna
- 2: Boeing
- 3: Caça
