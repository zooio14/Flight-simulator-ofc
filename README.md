# Flight Simulator 1 — Versão 0.2 Mundo Aberto

Versão melhorada feita para rodar no navegador e no iPad.

## Novidades

- Mundo aberto sem paredes invisíveis
- Terreno procedural infinito por chunks
- Montanhas, florestas e cidades geradas automaticamente
- Aeroporto inicial com pista, hangares e torre
- Avião controlável em 3D
- Câmera em 3ª pessoa, cockpit e cinema
- Controles por teclado e toque
- Sistema simples de combustível
- Decolagem, pouso e voo livre

## Controles

- W: acelerar
- S: frear/reduzir potência
- Setas: controlar o avião
- A/D: virar
- C: trocar câmera
- R: resetar avião

No iPad, use os botões na tela.

## GitHub Pages

Suba `index.html`, `style.css`, `main.js` e `README.md` no repositório. Depois vá em Settings → Pages → Deploy from a branch → main → /root.
