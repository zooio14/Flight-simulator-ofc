# Flight Simulator 1 v0.4 Realista

Versão melhorada para navegador/iPad.

## Melhorias

- Mundo aberto sem paredes invisíveis.
- Terreno infinito por chunks.
- Texturas procedurais mais realistas.
- Cidades, florestas, montanhas e oceano.
- Aeroporto maior.
- Avião controlável.
- Câmera externa, cockpit e cinema.
- Controles por toque para iPad.
- Código preparado para evoluir com mapas reais/satélite depois.

## Importante sobre texturas reais

Texturas reais de satélite exigem API externa, como Mapbox, Google Maps ou Cesium.
Esta versão usa texturas procedurais gratuitas para funcionar direto no GitHub Pages.

## Como publicar no GitHub Pages

Suba estes arquivos no repositório:

- index.html
- style.css
- main.js
- README.md

Depois vá em Settings > Pages > Deploy from branch > main > root.
