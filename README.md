# Flight Simulator OFC V3 Funcionando

Versão corrigida.

## Correção principal

O avião agora acelera de verdade.

Na versão anterior, a física estava realista demais e o motor parecia fraco. Nesta versão, o motor foi ajustado para ser jogável e decolar melhor.

## Como decolar

1. Segure W.
2. Espere chegar perto de 120 km/h.
3. Aperte ↑ aos poucos.
4. Não puxe demais para não entrar em stall.
