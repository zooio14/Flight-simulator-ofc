# FLIGHT-SIMULATOR — Protótipo Jogável

Este é um protótipo jogável em navegador usando HTML, CSS, JavaScript e Three.js.

## Como jogar

Abra `index.html` no navegador.

Controles:

- W: aumentar potência
- S: reduzir potência
- Seta para cima: levantar o nariz
- Seta para baixo: baixar o nariz
- Seta esquerda/direita: inclinar
- A/D: virar
- R: resetar

## Como publicar no GitHub Pages

1. Envie estes arquivos para o repositório.
2. Vá em Settings.
3. Clique em Pages.
4. Em Branch, escolha `main`.
5. Em folder, escolha `/root`.
6. Clique em Save.
7. Aguarde o link do GitHub Pages aparecer.

Depois disso o jogo roda direto pelo link do GitHub Pages.
